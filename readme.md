# Cursor Landing Page Recreation

This project is a recreation of the Cursor AI coding tool's official landing page, built using HTML, CSS, and custom fonts.

## Parts Recreated

The following sections from the original Cursor landing page have been recreated:

1. **Navigation Bar** - Logo, navigation links (Features, Enterprise, Pricing, Resources), and action buttons (Sign in, Download)

2. **Hero Section** - Main headline "Built to make you extraordinarily productive, Cursor is the best way to code with AI" with download button and hero image

3. **Trusted Logos Section** - "Trusted every day by millions of professional developers" with company logos (Stripe, OpenAI, Linear, DataDog, NVIDIA, Figma, Ramp, Adobe)

4. **Features Blocks** - Three feature sections:
   - "Agent turns ideas into code"
   - "Magically accurate autocomplete"
   - "Everywhere software gets built"

5. **Testimonials Section** - "The new way to build software" with testimonial cards from Diana Hu (Y Combinator), shadcn, Andrej Karpathy (Eureka Labs), Patrick Collison (Stripe), ThePrimeagen, and Greg Brockman (OpenAI)

6. **Stay on the Frontier Section** - Three cards about accessing models, codebase understanding, and enterprise features

7. **Changelog Section** - Recent version updates and features

8. **Team Section** - "Cursor is an applied team focused on building the future of coding" with join button

9. **Recent Highlights** - Blog-style posts about product updates and research

10. **Footer** - Comprehensive footer with product links, resources, company info, legal links, and social connections

## Fonts Used

- **Primary Font**: Cursor Gothic (Custom font family)
  - Loaded from local `fonts/` directory
  - Font Awesome icons for various UI elements

## Colors Used

The color scheme follows a dark theme with the following key colors:

- **Background**: `#100E09` (Dark brown/black)
- **Container Background**: `#16140F` (Slightly lighter dark)
- **Text Color**: `#DCD9D5` (Light gray)
- **Button Background**: `#EDECEC` (Light gray)
- **Button Text**: `#14120B` (Dark brown/black)
- **Hover Effects**: Various opacity changes for interactive elements
